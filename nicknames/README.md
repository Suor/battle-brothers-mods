# Nicknames for Everyone

Gives every brother a nickname (title) that fits who he is. A disloyal one gets "Rat", a tiny quick-witted fellow gets "Imp", someone good with dagger gets "Stinger" and so on. No random fluff — each nickname is earned by background, traits, attributes, talents, perks, injuries or their combination. It is not always straightforward though, some nicknames can be literal or ironic or metaphoric, they also can be all of the above on different occasions, so it may be used as hint for hiring but an unreliable one.

So it has:

- large pool of nicknames, each tied to specific factors
- avoids reusing nicknames already taken by living or fallen brothers
- integrates with vanilla trait/background titles lists
- plays nice with other mods altering bros:
    - [Bro Studio][bro_studio]
    - [Hackflow's Perks][hackflows_perks]
    - [Starting Perks By Background (Updated)][mod_background_perks]
    - [Elite Few - Master Bros (Updated)][elite_few]
- knows about many mods adding backgrounds and traits:
    - [More Backgrounds and Events (Updated)][xbe]
    - Reforged
    - Legends
    - Fantasy Bros
    - Northern Expansion
    - HeroicScenarioPack
    - [Proper Necro][necro]
    - Lone Chosen
    - Cultist Origin

Russian localization included - need to install [rosetta][] and it's dependencies.


## Getting the Most

Certain mods and their settings will open more factors and thus more diverse titles. This is not required but will make things more fun by triggering more nicknames, but also on it's own.

1. [More Backgrounds and Events (Updated)][xbe] - more backgrounds to mock and praise, and to play.
2. [Starting Perks By Background (Updated)][mod_background_perks] - starting perks also affect nicknames.
3. [Bro Studio][bro_studio] - set 1 extra starting trait for each bro. Will make combo nicknames trigger more often.


## Compatibility

Safe to add or remove midgame. Should be compatible with everything.


## Install

Download a zip from [NexusMods][] and put it into `data` dir. Requires [ModernHooks][] and [stdlib][].

To use built-in russian translation will also need to install [Rosetta][] and the translation of the game itself, i.e.data_014 and fonts. Once this is done the mod will switch to Russian automatically.
